# Summer of CCNA

Welcome to my **Summer of CCNA** learning repository.

This repo documents my journey through NetworkChuck's Summer of CCNA course as I build foundational networking skills and work toward breaking into the IT field.

## About This Repository

The purpose of this repository is to track my progress, document what I am learning, and showcase hands-on networking practice through notes, labs, troubleshooting exercises, and Packet Tracer projects.

I am using this repo as a public learning portfolio to demonstrate consistency, technical growth, and practical understanding of networking concepts.

## Goals

- Build a strong foundation in computer networking
- Learn and practice core CCNA concepts
- Develop hands-on experience with Cisco Packet Tracer
- Improve troubleshooting skills
- Document my learning in a clear and professional way
- Build a portfolio that supports my transition into IT

## Topics I Am Studying

- Network fundamentals
- OSI and TCP/IP models
- Ethernet and switching
- IPv4 addressing and subnetting
- VLANs and trunking
- Inter-VLAN routing
- Static routing
- Dynamic routing concepts
- DHCP
- NAT
- ACLs
- Wireless basics
- Network security fundamentals
- Automation and programmability basics
